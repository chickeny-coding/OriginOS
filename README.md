## OriginOS

### Features

* Print `OriginOS` on screen.

### Usages

* Clone the repository from GitHub.

```bash
git clone git@github.com:chickeny-coding/OriginOS
```

* Move into the repository.

```bash
cd OriginOS/
```

* Make the repository (make sure you have `make`).

```bash
make
```

* Simulate OriginOS (make sure you have `qemu-system-x86_64`).

```bash
make qemu
```
